<h1>Projeto</h1>

Projeto Todolist foi criado utilizando os conceitos de lista, realizando componentização e estilização com styled-components.<br/>
Utilização de tipagem com typescript.<br/>
Projeto foi realizado na semana D5 ReactJS da B7web.

<h1>Tecnologia</h1>

- [ReactJS](https://pt-br.reactjs.org/)
- [Typescript](https://www.typescriptlang.org/)

<h1>Instalação</h1>

**Requisitos**
- clone o repositório e instale as dependências yarn ou npm<br/>
- Ou acesse [todo-list](https://marceloasouza.github.io/todo-list/)
